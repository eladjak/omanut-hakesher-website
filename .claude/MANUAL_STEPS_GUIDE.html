4

חיבור Google Analytics 4

כדי לעקוב אחרי מבקרים, מקורות תנועה, ודפים פופולריים

ℹ️

מה זה GA4?

Google Analytics 4 הוא כלי חינמי של גוגל שמראה לך כמה אנשים ביקרו באתר, מאיפה הגיעו (גוגל, פייסבוק, ישיר), אילו דפים הם קראו, ועוד. חיוני לכל עסק.

1

כנס ל-Google Analytics

פתח דפדפן וכנס לכתובת:

https://analytics.google.com

היכנס עם חשבון הגוגל שלך (אותו חשבון שאתה משתמש לעסק).

2

צור Property חדש (אם אין לך כבר)

בתפריט השמאלי ← לחץ על שם החשבון ← "Create Property".

מה למלא: Property name: אומנות הקשר
Reporting time zone: Israel
Currency: Israeli Shekel (ILS)

לחץ "Next" ← Business size: "Small" ← Business objectives: "Generate leads" ← "Create".

3

הוסף Web Stream

אחרי יצירת ה-Property ← בחר "Web" ← הכנס:

פרטי ה-stream: Website URL: https://omanut-hakesher-website.vercel.app
(בהמשך תשנה לדומיין הסופי שלך)
Stream name: אומנות הקשר - אתר ראשי

לחץ "Create stream".

4

העתק את ה-Measurement ID

אחרי יצירת ה-stream, תראה מסך עם פרטי ה-stream. חפש את Measurement ID - הוא נראה כך:

G-XXXXXXXXXX

לחץ על הסמל להעתקה לצד ה-ID. שמור אותו, תצטרך אותו בשלב הבא.

⚠️

שים לב! ה-ID מתחיל ב-G- ולא ב-UA-. אם אתה רואה UA- זה GA3 הישן - תצטרך לצור property חדש.

5

הוסף את ה-ID ל-Vercel

עכשיו יש להוסיף את ה-ID כ-environment variable ב-Vercel. כנס ל:

https://vercel.com/eladjaks-projects/omanut-hakesher-website/settings/environment-variables

לחץ "Add New" ← מלא:

Name: NEXT_PUBLIC_GA_ID Value: G-XXXXXXXXXX ← ה-ID שהעתקת Environment: Production, Preview, Development

לחץ "Save".

6

שלח לי את ה-ID

שלח לי את ה-Measurement ID (G-XXXXXXXXXX) בצ'אט. אני אוסיף את קוד ה-tracking לאתר בדיוק כמו שצריך (Google Tag Manager style עם gtag.js).

💡

למה שלח לי ולא לשנות לבד?

קוד ה-Analytics צריך להתווסף ל-layout.tsx בצורה מדויקת שתעבוד עם Next.js App Router, עם Server Components, ועם ה-GDPR compliance. יש לזה pattern ספציפי שאני אטמיע.

7

אמת שהכל עובד

אחרי שאטמיע את הקוד ואדפלוי:

  • כנס ל-analytics.google.com
  • בחר את ה-Property שיצרת
  • לחץ "Reports" ← "Realtime"
  • פתח את האתר בטאב אחר
  • אתה אמור להופיע כ-"1 active user" תוך דקה-שתיים

מה תוכל לראות ב-Analytics?

ממאיפה מגיעים המבקרים (גוגל? פייסבוק? ישיר?) • אילו דפים הם קוראים • כמה זמן הם מבלים • מה המסלול שלהם (בית ← coaching ← contact) • ממה מגיעים הכי הרבה לידים. כל זה חינם!

1. Google Analytics — Measurement ID
חסום
עדיפות גבוהה

האתר מוכן לאנליטיקס אבל חסר ה-Measurement ID. בלי זה אין נתונים על תנועה.

  1. כנס ל-analytics.google.com
  2. צור חשבון חדש (או השתמש בקיים) → Property חדש בשם "אומנות הקשר"
  3. בחר Web → הכנס את הכתובת omanut-hakesher.co.il
  4. העתק את ה-Measurement ID (נראה כך: G-XXXXXXXXXX)
  5. שלח לי את ה-ID ואני אשלב אותו באתר
2. חיבור דומיין ohlove.co.il
חסום
עדיפות גבוהה

האתר עולה ב-vercel.app אבל צריך את הדומיין האמיתי. זה דורש גישה לספק הדומיין שלך.

  1. כנס לאיפה שקנית את הדומיין (iWantMyName? Namecheap? אחר?)
  2. כנס ל-vercel.com → הפרויקט → Settings → Domains
  3. הוסף ohlove.co.il ו-www.ohlove.co.il
  4. Vercel יציג לך רשומות DNS שצריך להוסיף אצל ספק הדומיין
  5. הוסף את רשומות ה-CNAME/A בהגדרות ה-DNS של הדומיין
  6. חכה 5-30 דקות עד שהדומיין מתחבר (Vercel יראה סטטוס ירוק)
לא בטוח מי ספק הדומיין?

נסה לחפש במייל שלך "ohlove.co.il" — בדרך כלל יש מייל חידוש שנתי מהספק.

3. חיבור דפי Lead Magnets ל-Responder
ממתין
עדיפות בינונית

שלושת דפי ה-Lead Magnets מוכנים ויזואלית. כדי שהם ירשמו אנשים לרשימת התפוצה, צריך לחבר ל-Responder.

  1. כנס ל-Responder
  2. צור 3 רשימות חדשות: "23 סיבות", "7 עקרונות", "36 שאלות"
  3. לכל רשימה, צור טופס הרשמה וקבל את ה-API key / Form URL
  4. שלח לי את הפרטים ואני אחבר את הטפסים באתר

חלופה: אם אין לך חשבון Responder, אפשר להשתמש ב-Mailchimp (חינם עד 500 נרשמים) או ConvertKit.

4. ייצוא תוכן מ-RavMeser
חסום
עדיפות בינונית

לאתר RavMeser הישן יש תוכן שיכול להעשיר את האתר החדש. Cloudflare חוסם גישה אוטומטית, אז צריך ייצוא ידני.

  1. כנס לאדמין של RavMeser (רבמסר) עם הסיסמה שלך
  2. ייצא את התכנים הבאים:
    • מאמרים / פוסטים
    • דפי נחיתה
    • רשימת נרשמים (CSV)
  3. שמור את הקבצים בתיקייה ב-OneDrive ותגיד לי

מדריך מפורט נמצא בקובץ: .claude/MANUAL_STEPS_GUIDE.html
